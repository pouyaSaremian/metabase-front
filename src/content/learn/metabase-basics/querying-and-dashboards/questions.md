# پرسیدن سؤال‌ها

نحوه استفاده از query builder متابیس برای پرسیدن سؤال‌های بهتر.

#### [ایجاد نمودارهای تعاملی](questions/drill-through.html)
نمودارهایی بسازید که مردم می‌توانند با استفاده از query builder، ساخت مدل‌ها، یا افزودن مقصدهای سفارشی کاوش کنند.
4 دقیقه مطالعه

#### [آموزش: عبارات سفارشی](questions/custom-expressions.html)
چه زمانی باید از عبارات سفارشی استفاده کنید و چرا باید از آن‌ها در ویرایشگر notebook متابیس استفاده کنید.
7 دقیقه مطالعه

#### [آموزش: Joinها در متابیس](questions/joins-in-metabase.html)
نحوه join کردن جداول در متابیس با استفاده از ویرایشگر notebook در سؤال‌های ساده و سفارشی.
8 دقیقه مطالعه

#### [ترکیب داده از پایگاه‌های داده مختلف](questions/cross-db-joins.html)
نحوه join کردن جداول از پایگاه‌های داده مختلف در متابیس.
7 دقیقه مطالعه

#### [آموزش: تجمیع چندسطحی](questions/multi-aggregation.html)
نحوه پرسیدن سؤال‌ها با چندین بخش با استفاده از query builder.
2 دقیقه مطالعه

#### [آموزش: جستجو در جداول](questions/searching-tables.html)
یاد بگیرید چگونه در سؤال‌های SQL و سؤال‌های ساده خود با استفاده از فیلترها و عبارات سفارشی جستجو کنید. پیدا کردن کلمات یا عبارات در جداول شما اکنون آسان‌تر از همیشه است.
5 دقیقه مطالعه

#### [آموزش: تمیز و فرمت کردن متن](questions/formatting.html)
نحوه استفاده از عبارات سفارشی برای تمیز کردن متنی که ناسازگار، بدون ساختار، یا خالی است.
7 دقیقه مطالعه
