# حریم خصوصی

## آیا برای انطباق با GDPR به توافق‌نامهٔ پردازش داده (DPA) با متابیس نیاز دارید؟

### استقرار self‑hosted

اگر متابیس را به‌صورت self‑hosted اجرا می‌کنید، ما **هیچ دسترسی مستقیمی به داده‌های شما نداریم**، مگر این‌که شما به‌طور صریح گزینهٔ ارسال [آمار استفادهٔ ناشناس][information-collection] را فعال کرده باشید. حتی در آن صورت هم متابیس هیچ‌گونه دادهٔ هویتی قابل‌شناسایی (PII) جمع‌آوری نمی‌کند. بنابراین، زمانی که متابیس را خودتان میزبانی می‌کنید، از منظر GDPR ما «پردازش‌کنندهٔ داده» (Data Processor) محسوب نمی‌شویم.

### Metabase Cloud

اگر از سرویس [Metabase Cloud](https://www.metabase.com/pricing/) استفاده می‌کنید، [شرایط استفادهٔ شما](https://www.metabase.com/license/hosting) حاوی یک توافق‌نامهٔ پردازش داده (Data Processing Agreement یا DPA) است.
