# کلیدهای میانبر

فهرستی کاربردی از کلیدهای میانبر که به شما کمک می‌کند سریع در متابیس جابه‌جا شوید.

کلیدهای میانبر بسته به صفحه‌ای که در متابیس روی آن هستید کمی متفاوت‌اند.

برای دیدن منوی کلیدهای میانبر در خود متابیس، دکمهٔ `?` را بزنید، یا روی آیکون **چرخ‌دنده (Gear)** در گوشهٔ بالا-راست کلیک کنید و گزینهٔ **Keyboard shortcuts** را انتخاب کنید.

## حالت کلی (Global)

| عمل                        | میانبر       |
| -------------------------- | ------------ |
| ساخت سؤال                  | c > q        |
| ساخت سؤال native           | c > n        |
| ساخت داشبورد               | c > d        |
| ساخت کالکشن                | c > f        |
| ساخت مدل                   | c > m        |
| ساخت Metric                | c > k        |
| مرور پایگاه‌داده‌ها        | g > d        |
| مرور مدل‌ها                | g > m        |
| مرور Metricها              | g > k        |
| گزارش مشکل                 | Ctrl/Cmd+f1  |
| مشاهدهٔ شورت‌کات‌ها        | ?            |
| باز کردن Trash             | g > t        |
| باز کردن کالکشن شخصی       | g > p        |
| باز و بسته‌کردن سایدبار    | [            |
| رفتن به صفحهٔ ادمین        | g > a        |
| رفتن به تنظیمات کاربری     | g > u        |
| رفتن به صفحهٔ خانه         | g > h        |

| عمل                            | میانبر             |
| ------------------------------ | ------------------ |
| بوکمارک‌کردن داشبورد          | o                  |
| افزودن فیلتر                   | f                  |
| باز کردن سایدبار سؤالات       | a                  |
| افزودن سؤال با Query builder  | q                  |
| افزودن سؤال native             | n                  |
| لغو ویرایش داشبورد            | e                  |
| فرستادن داشبورد به Trash      | Ctrl/Cmd+backspace |
| نمایش/مخفی‌کردن اطلاعات داشبورد | ]                |
| ویرایش داشبورد                | e                  |
| ذخیرهٔ داشبورد                | s                  |
| عوض‌کردن تب داشبورد           | 1، 2، 3، ...       |
