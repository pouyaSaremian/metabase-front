# متابیس کلود

«متابیس کلود» سرویس میزبانی‌شدهٔ متابیس برای پلن‌های [Starter](https://www.metabase.com/product/starter)، [Pro](https://www.metabase.com/product/pro) و [Enterprise](https://www.metabase.com/product/enterprise) است. در این سرویس، تیم متابیس زیرساخت سرور شما را مدیریت می‌کند، در دسترس‌بودن (uptime) و پشتیبان‌گیری‌ها را برعهده می‌گیرد و به‌روزرسانی‌ها را انجام می‌دهد تا همیشه آخرین نسخهٔ متابیس را در اختیار داشته باشید.

## مستندات متابیس کلود

برای مدیریت اینستنس متابیس کلود، اشتراک و صورت‌حساب‌ها، از [Metabase Store](https://store.metabase.com/account) استفاده کنید.
