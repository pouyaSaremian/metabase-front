# نمودارهای ترکیبی (Combo)

نمودارهای Combo به شما اجازه می‌دهند Bar و Line (یا Area) را روی یک نمودار ترکیب کنید.

![Line + bar](../images/combo-chart.png)

متابیس به‌طور پیش‌فرض یکی از سری‌ها را به‌صورت Line و دیگری را به‌صورت Bar نمایش می‌دهد. می‌توانید با بازکردن تنظیمات Visualization مشخص کنید کدام سری Line، کدام Bar و کدام Area باشد، و تنظیمات جداگانه‌ای مثل رنگ‌ها را برای هر سری اعمال کنید.

برای استفاده از Combo chart، یا باید در بخش Summarize By سؤال‌تان دست‌کم دو Metric و یک یا دو ستون Grouping داشته باشید، مثل این:

![Data for Line + Bar chart](../images/combo-chart-data-1.png)
