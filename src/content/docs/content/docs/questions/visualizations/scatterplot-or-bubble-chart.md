# نمودارهای Scatter و Bubble

**Scatterplot**‌ها برای نمایش رابطه یا هم‌بستگی بین دو متغیر مفید هستند؛ مثلاً مقایسهٔ سن کاربران با مبلغی که برای محصولات شما خرج کرده‌اند. برای استفاده از Scatterplot باید سؤالی بپرسید که خروجی‌اش دو ستون عددی داشته باشد، مثل `Count of Orders grouped by Customer Age`. همچنین می‌توانید از یک Table استفاده کنید و در تنظیمات نمودار دو فیلد عددی موردنظر را انتخاب کنید.

اگر یک فیلد عددی سوم هم داشته باشید، می‌توانید یک **Bubble chart** بسازید. Visualization نوع Scatter را انتخاب کنید، سپس تنظیمات نمودار را باز کرده و در Dropdown مربوط به **bubble size**، فیلدی را انتخاب کنید که اندازهٔ هر Bubble را تعیین می‌کند. برای نمونه، می‌توانید از فیلدی استفاده کنید که کل مبلغ سفارش برای هر جفت x-y را نگه می‌دارد — یعنی هرچه مبلغ کل بیشتر باشد، Bubble بزرگ‌تر خواهد بود.

Scatter و Bubble از نظر تنظیمات نمودار شبیه Line، Bar و Area هستند و می‌توانید روی آن‌ها Trend line یا Goal line هم نمایش دهید.
