# متابوت - دستیار هوش مصنوعی متابیس

{% include beta-blockquote.html message="در حال حاضر، متابوت فقط به‌عنوان افزونه‌ای برای پلن‌های Pro و Enterprise در متابیس کلود در دسترس است." %}

![آشنایی با متابوت](./images/metabot.png)

متابوت به شما کمک می‌کند داده‌هایتان را تحلیل کنید: از ساخت نمودار بر اساس زبان طبیعی و تولید کوئری‌های SQL، تا رفع خطاهای کوئری و تحلیل بصری‌سازی‌های موجود.

برای راه‌اندازی متابوت، به [تنظیمات متابوت](./settings.md) مراجعه کنید.

## متابوت چه کارهایی می‌تواند انجام دهد؟

متابوت می‌تواند به شما کمک کند:

- [با استفاده از query builder از روی زبان طبیعی نمودار بسازید](#نحوهٔ-استفادهٔ-متابوت-از-query-builder).
- در [ویرایشگر SQL](../questions/native-editor/writing-sql.md) از روی زبان طبیعی SQL تولید کنید (در حال حاضر فقط SQL پشتیبانی می‌شود).
- [یک نمودار را تحلیل کنید](#تحلیل-نمودارها-با-متابوت).
- [خطاهای موجود در SQL را برطرف کنید](#رفع-خطاهای-sql-با-کمک-متابوت).
- از مستندات (همین محتوایی که الآن می‌خوانید) سؤال بپرسید و پاسخ بگیرید.

مثل هر سیستم هوش مصنوعی مولد، همیشه باید خروجی متابوت را دوباره بررسی کنید.

## نوار کناری چت متابوت

![نوار کناری چت متابوت](./images/metabot-conversation-sidebar.png)

چند راه مختلف برای شروع چت با متابوت وجود دارد:

- روی مک دکمه‌های cmd+e و روی ویندوز ctrl+e را بزنید تا [نوار کناری چت](#نوار-کناری-چت-متابوت) باز شود.
- روی آیکون متابوت در گوشهٔ بالا-راست کلیک کنید.

می‌توانید با متابوت چت کنید (البته همان‌طور که انتظار می‌رود، فقط در مورد کمک به تحلیل داده‌ها علاقه‌مند است).

متابوت زمینهٔ (context) سؤال فعلی را در طول یک گفت‌وگو حفظ می‌کند. فقط تاریخچهٔ مکالمهٔ جاری ذخیره می‌شود (می‌توانید برای دیدن پیام‌های قبلی به بالا اسکرول کنید). اگر یک چت جدید شروع کنید، متابیس مکالمهٔ قبلی را کنار می‌گذارد، پس هنگام ریست کردن مکالمه دقت کنید.

چند نکته:

- تا جایی که می‌توانید به متابوت کانتکست بدهید. اگر جدول یا فیلدهای مدنظر خود را می‌دانید، آن‌ها را به متابوت بگویید.
- هر وقت می‌خواهید متابوت کار کاملاً متفاوتی انجام دهد، بهتر است مکالمه را ریست کنید، چون کانتکست قدیمی ممکن است متابوت را سردرگم کند.
- وقتی متابوت برای شما یک سؤال (کوئری) می‌سازد، می‌توانید با پرسش‌های بعدی ادامه دهید یا خودتان کنترل را به‌دست بگیرید. می‌توانید روی نمودار drill کنید یا وارد ویرایشگر شوید و کوئری را (در query builder یا ویرایشگر SQL) ویرایش کنید.
- متابوت با پرامپت‌های انگلیسی بهترین عملکرد را دارد. ممکن است زبان‌های دیگر را هم تا حدی بفهمد، اما مطمئن‌ترین نتایج را با پرسیدن سؤالات به انگلیسی می‌گیرید.

### منوی پاسخ متابوت

روی پاسخ متابوت که موس را نگه دارید می‌توانید:

- پاسخ را کپی کنید.
- برای پاسخ‌ها [بازخورد مثبت/منفی بدهید](#ارسال-بازخورد-برای-پاسخهای-متابوت) (thumbs-up/thumbs-down). اگر پاسخ را تأیید کنید (thumbs up)، می‌توانید به‌صورت اختیاری توضیحی اضافه کنید. اگر آن را رد کنید (thumbs down)، می‌توانید یک گزارش خطا ثبت کنید تا به بهبود متابوت کمک کنید.
- همان پرامپت را دوباره اجرا کنید. این کار وقتی مفید است که نمودار را به‌روزرسانی کرده‌اید یا فقط می‌خواهید متابوت یک پاسخ جدید (با مدل) تولید کند؛ چون پاسخ‌های AI تعیین‌ناپذیر (non-deterministic) هستند، متابوت ممکن است در بار بعد خروجی متفاوتی بدهد.

## تحلیل نمودارها با متابوت

![متابوت در حال تحلیل نمودار](./images/metabot-response.png)

وقتی یک سؤال (پرسش) را مشاهده می‌کنید، می‌توانید روی آیکون متابوت در گوشهٔ بالا-راست کلیک کنید تا نمودار تحلیل شود. همچنین می‌توانید command palette را باز کنید و از متابوت بخواهید نمودار را تحلیل کند.

اگر خروجی را به‌صورت جدول نتایج می‌بینید، ممکن است دکمهٔ متابوت روی نمودار نمایش داده نشود، اما همچنان می‌توانید چت متابوت را باز کنید و از او بخواهید جدول را تحلیل کند؛ در این حالت متابوت یک [X-ray](../exploration-and-organization/x-rays.md) از نتایج می‌سازد.

همچنین می‌توانید از متابوت بخواهید دربارهٔ جدول‌های مشخصی در پایگاه‌داده به شما توضیح بدهد.

## نحوهٔ استفادهٔ متابوت از query builder

وقتی از متابوت می‌خواهید از روی زبان طبیعی نمودار بسازد، ابتدا به‌دنبال سؤالات (پرسش‌های) موجودی می‌گردد که شاید بتوانند پاسخ شما را بدهند. اگر سؤال مناسبی پیدا کند، به‌جای ساختن نمودار جدید، همان را به شما پیشنهاد می‌کند. در غیر این صورت، متابوت از [query builder](../questions/query-builder/editor.md) برای ساخت نمودار جدید استفاده می‌کند.

در نظر داشته باشید که متابوت هنوز در حال یادگیری query builder است. متابوت فقط می‌تواند عملیات سادهٔ query builder را انجام دهد و به کتابخانهٔ [عبارات سفارشی](../questions/query-builder/expressions-list.md) دسترسی ندارد. متابوت همچنین به تجمیع و گروه‌بندی در یک سطح محدود است؛ بنابراین اگر به تحلیل پیچیده‌تری نیاز دارید، می‌توانید کوئری را خودتان ویرایش و تکمیل کنید یا به ویرایشگر SQL بروید.

## متابوت در ویرایشگر SQL

![متابوت SQL را از روی پرامپت زبان طبیعی تولید می‌کند](./images/generate-sql-from-natural-language-prompt.png)

برای این‌که متابوت برای شما SQL بنویسد:

1. [ویرایشگر SQL](../questions/native-editor/writing-sql.md) را باز کنید.
2. پایگاه‌داده‌ای را که می‌خواهید روی آن کوئری بزنید انتخاب کنید.
3. روی مک cmd+e و روی ویندوز ctrl+e را بزنید تا [نوار کناری چت](#نوار-کناری-چت-متابوت) باز شود.
4. چیزی شبیه «یک کوئری SQL بنویس که...» بنویسید و پرامپت خود را کامل کنید.

متابوت SQL را برای شما تولید می‌کند، اما آن را اجرا نمی‌کند. این کار به شما فرصت می‌دهد قبل از اجرا، کوئری را بررسی و در صورت نیاز اصلاح کنید.

اگر در پرامپت خود جدول مشخصی را نام نبرید، متابوت فقط ۱۰۰ جدول اول در پایگاه‌دادهٔ انتخاب‌شده را بررسی می‌کند. اگر سؤال شما به جدول‌هایی غیر از این ۱۰۰ جدول اول مربوط باشد، متابوت ممکن است جدول‌های لازم را «حدس بزند» (hallucinate) و در نتیجه کوئری شکست بخورد.

## رفع خطاهای SQL با کمک متابوت

![متابوت می‌تواند سعی کند خطاهای SQL را اصلاح کند](./images/have-metabot-fix-it.png)

وقتی در اجرای یک کوئری SQL خطا می‌گیرید، می‌توانید روی دکمهٔ **Have Metabot fix it** کلیک کنید تا متابوت سعی کند کوئری را اصلاح کند. همچنین می‌توانید در چت از متابوت بخواهید SQL شما را درست کند.

## ناوبری بعد از ساخت نمودار توسط متابوت

اگر متابوت یک کوئری بسازد یا شما را به آیتم جدیدی ببرد و بخواهید به صفحهٔ قبلی برگردید، می‌توانید از دکمهٔ Back مرورگر خود استفاده کنید.

همچنین می‌توانید هر نموداری را که متابوت می‌سازد ذخیره کنید و آن را به یک داشبورد یا کالکشن اضافه کنید.

## ارسال بازخورد برای پاسخ‌های متابوت

وقتی روی پاسخ‌های متابوت هاور می‌کنید، گزینه‌هایی برای ارسال بازخورد می‌بینید.

بازخورد مثبت (thumbs up) به‌اندازهٔ بازخورد منفی مفید است. اگر thumbs down بدهید، می‌توانید به‌صورت اختیاری جزئیات بیشتری دربارهٔ مشکل و نوع آن ارائه کنید:

- UI bug
- Took incorrect actions
- Overall refusal
- Did not follow request
- Not factually correct
- Incomplete response
- Other

> وقتی بازخورد ارسال می‌کنید، فرمی که می‌فرستید ممکن است شامل داده‌های حساسی از مکالمهٔ شما با متابوت باشد.
