# نمای کلی سؤالات

سؤال‌ها ترکیبی از کوئری و ویژوالیزیشن آن هستند. می‌توانید با استفاده از Query builder گرافیکی متابیس سؤال بپرسید، یا یک کوئری Native/SQL بنویسید.

## [مقدمهٔ سؤال‌ها](./introduction.md)

سؤال‌ها واحدهای سازندهٔ اصلی متابیس هستند.

## Query builder

### [ویرایشگر کوئری](./query-builder/editor.md)

مقدمه‌ای بر ویرایشگر گرافیکی کوئری در متابیس.

### [عبارت‌های سفارشی](./query-builder/expressions.md)

عبارت‌های سفارشی، شبیه فرمول‌های صفحه‌گسترده، ابزارهایی هستند که می‌توانید در Query builder گرافیکی از آن‌ها استفاده کنید تا سؤال‌های پیچیده‌تری بپرسید.

### [فهرست عبارت‌ها](./query-builder/expressions-list.md)

فهرستی از Aggregationها و توابعی که در Query builder گرافیکی در دسترس‌اند.

### [جوین کردن داده‌ها](./query-builder/join.md)

می‌توانید داده‌ها را جوین کنید تا دادهٔ فعلی‌تان را با یک جدول دیگر یا حتی یک سؤال ذخیره‌شده ترکیب کنید.

## ویرایشگر کوئری Native

همان SQL editor است (می‌گوییم Native چون می‌توانید روی پایگاه‌داده‌هایی که لزوماً SQL نیستند، مثل MongoDB، هم کوئری بزنید).

### [مقدمه‌ای بر ویرایشگر کوئری Native](./native-editor/writing-sql.md)

کد Native (مثل SQL) برای منبع داده‌تان بنویسید.

### [الگوهای SQL](./native-editor/sql-parameters.md)

پارامترها را به‌صورت متغیر در Templateهای SQL خود تزریق کنید.

### [Field filterها](./native-editor/field-filters.md)

با متصل‌کردن متغیرهای Field filter به فیلدهای پایگاه‌داده، ویجت‌های فیلتر هوشمند بسازید.

### [پارامترهای پایهٔ SQL](./native-editor/basic-sql-parameters.md)

با اصول استفاده از پارامترها در کوئری‌های SQL آشنا شوید.

### [متغیرهای اختیاری](./native-editor/optional-variables.md)

با استفاده از براکت‌ها، پارامترهایی بسازید که می‌توانند در کوئری خالی بمانند.

### [پارامترهای گروه‌بندی زمانی](./native-editor/time-grouping-parameters.md)

با استفاده از پارامترها در کوئری‌های Native، داده را بر اساس بازه‌های زمانی گروه‌بندی کنید.

### [ویجت‌های فیلتر](./native-editor/filter-widgets.md)

ویجت‌های فیلتری را تنظیم کنید که به متغیرهای موجود در کوئری‌های Native متصل هستند.

### [ارجاع به سؤال‌های ذخیره‌شده در کوئری‌ها](./native-editor/referencing-saved-questions-in-queries.md)

از سؤال‌های ذخیره‌شده به‌عنوان منبع داده در کوئری‌های Native استفاده کنید.

### [Snippets](./native-editor/snippets.md)

تکه‌های SQL را دوباره استفاده و با دیگران به‌اشتراک بگذارید.

## اشتراک‌گذاری نتایج

راه‌های مختلفی برای اشتراک‌گذاری نتایج سؤال‌ها وجود دارد. برای اشتراک‌گذاری نتایج یک داشبورد، به [اشتراک‌های داشبورد](../dashboards/subscriptions.md) مراجعه کنید.

### [ویژوالیزه‌کردن نتایج](./visualizations/visualizing-results.md)

از بین انواع مختلف ویژوالیزیشن انتخاب کنید.

### [هشدارها](./alerts.md)

نتایج را ازطریق ایمیل یا Slack دریافت کنید، چه طبق زمان‌بندی، و چه فقط وقتی اتفاق خاصی می‌افتد.

### [Export کردن](./exporting-results.md)

راه‌های مختلفی که می‌توانید نتایج سؤال‌ها و داشبوردها را Export کنید.

### [Tooltipها](./visualizations/tooltips.md)

Tooltipها را روی نمودارهای ستونی، خطی و سطحی سفارشی کنید.
