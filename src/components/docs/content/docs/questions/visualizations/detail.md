# Detail

Visualization نوع **Detail** یک رکورد تکی (یک ردیف از نتیجهٔ کوئری) را در قالبی خوانا و دوستونه نمایش می‌دهد.

![Detail of a record in the account table](../images/detail.png)

می‌توانید با دکمه‌های فلش بین رکوردها جابه‌جا شوید.
